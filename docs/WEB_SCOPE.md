# Escopo Do AnalictY Web

O AnalictY Web e o salao da plataforma: a interface diaria da operacao industrial.

## Fica No Web

- Visao Geral.
- Status das maquinas.
- Historico de producao.
- Historico de paradas.
- Relatorios operacionais.
- Alertas operacionais.
- Telegram/notificacoes de operacao.
- Dashboards.
- Maquinas, TAGs, turnos, setores e centros de custo quando forem configuracoes operacionais.

## Deve Migrar Para O Manager

- Atualizacoes do sistema.
- Logs tecnicos.
- Diagnostico profundo do runtime.
- Reinicio de servicos.
- Backup e restauracao.
- Configuracao tecnica de banco de dados.
- Informacoes de manifesto/update.
- Ferramentas de suporte e manutencao.
- Auditoria tecnica pesada.

## Regra

Se a funcao e usada pela producao no dia a dia, fica no Web.

Se a funcao e de instalacao, manutencao, suporte ou administracao local, vai para o Manager.
